Stanford Javascript Crypto Library (SJCL)
Official Web Page / Source code @ Github

AES-256 in GCM mode with 128-bits tag

  • Auto-generated at run-time (on browser) by using the SJCL library's provided features
  • Generated key length: 50 chars in base64/url-safe format
  • The generated key is not directly used for encryption/decryption, but instead, it is first processed by a key derivation function (PBKDF2 with 100,000 iterations)

The server stores only the minimum information required from the encryption library to decrypt the message, based on the data that was originally used to encrypt it. This information consists of: IV, Salt and the encrypted message itself (CT: CipherText)
